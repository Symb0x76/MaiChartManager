## 功能

### 谱面管理

- 浏览本地谱面列表，和修改谱面基础信息
- 管理流派和版本分类的列表，支持自定义分类图片
- 检测和管理多个 Opt 目录之间的 ID 冲突

### 音视频处理

- 调整音频偏移、裁剪音频文件、设置预览片段
- 将视频转换为游戏兼容格式
- 替换或修改 ACB/AWB 音频文件

### 导入导出

- 导入 Simai/Maidata 谱面，自动进行格式转换
- 将谱面导出为 Opt 或 Maidata 格式
- 批量导出，支持按流派、版本分组

### [AquaMai](https://github.com/MuNET-OSS/AquaMai) 模组配置

- 一键安装和更新 AquaMai 模组
- 可视化配置编辑器，支持搜索和分类浏览
- 自定义触摸屏灵敏度，支持分区域精细调整
- 配置键盘/手台按键映射

### 其他工具

- 在软件内预览谱面
- 验证谱面数据，检查潜在问题
- 以服务器模式运行，通过浏览器远程访问

## 服务器模式

MaiChartManager 可以以服务器模式运行，允许通过浏览器远程管理谱面。适用于游戏电脑不便直接操作的场景。
